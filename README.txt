Maths Triangles
BOUROT Benjamin - PICHOT Bastien

Utilisation
=============
Pavé numérique : Changer de mode d'affichage
E : Passer en mode édition

En mode création
----------------
Clic gauche : Ajouter un point
N : Créer une nouvelle figure
C : Tout effacer 

En mode édition
----------------
Clic gauche : Sélectionner un point / Déplacer le point sélectionné
R : Supprimer le point sélectionné



Modes 
============= 
1 : Enveleoppe par marche de Jarvis
2 : Enveleoppe par Graham-Scan
3 : Trianculation quelconque
4 : Trianculation de Delaunay
5 : Diagrame de Voronoï
